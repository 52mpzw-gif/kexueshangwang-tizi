## 🛠️ 一、传统 Clash Verge Rev 的繁琐配置流程

要让这个软件跑起来，你通常需要折腾以下四个步骤：

1. **寻找并下载客户端：** 你需要去 GitHub 寻找对应的 Release 页面，在一堆 `msi`、`exe`、`zip` 针对不同 CPU 架构的安装包里挑出对的那一个。
2. **购买并导入订阅（最容易出错的一步）：** 你需要去买第三方的节点链接，复制一长串包含密钥的代码，粘贴到软件的 Profiles 里点击下载。如果遇到网络波动，经常会弹窗报错 "Download Failed"。
3. **手动选择分流与节点：** 导入后，面对密密麻麻的“全局”、“规则”、“直连”以及几十个国家和地区的节点，你需要自己去点闪电图标测试延迟，手动盲猜哪一个节点最稳定。
4. **手动开启系统代理：** 你必须在设置里手动勾选“系统代理”，软件才能真正接管流量。

对于只想安安稳稳查个资料、看个视频的用户来说，这种“程序员审美”的软件门槛实在是太高了。
